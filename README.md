## ✨ Overview

**Yarn Art Store** is a full-stack application built using the MERN stack (MongoDB, Express, React, Node.js).
It provides a seamless platform where users can **shop creative products** and **learn crafting skills** in one place.

---

## 🎯 Key Highlights

* 🧵 E-commerce platform for yarn & crochet products
* 📚 Integrated course system for learning DIY crafts
* 🔐 Secure authentication with JWT
* 🛠️ Admin dashboard for full control
* ⚡ Fast and responsive UI using React
* 🌐 Fully deployed (Vercel + Render)

---

## 🚀 Features

### 🛍️ User Features

* Browse yarn, crochet items, and DIY kits
* Add to cart and manage items
* Place and track orders
* Enroll in crafting courses

### 🔐 Admin Features

* Secure admin login
* Manage products (CRUD operations)
* Manage courses and content
* Handle users and orders

## ⚙️ Getting Started

### 🔧 Prerequisites

* Node.js installed
* MongoDB database (local or Atlas)

---

### 📥 Installation

```bash
git clone https://github.com/RiyaDwivedi12/yarnArtStore.git
cd yarn-art-store
```

---

### ⚙️ Backend Setup

```bash
cd backend
npm install
```

Create `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
RAZORPAY_KEY_ID=your_key
RAZORPAY_SECRET=your_secret
```

Run backend:

```bash
npm start
```

---

### 🎨 Frontend Setup

```bash
cd frontend
npm install
```

Create `.env` file:

```
VITE_API_URL=http://localhost:5000
```

Run frontend:

```bash
npm run dev
```

---

## 🚀 Deployment

* Frontend → Vercel
* Backend → Render

### ⚠️ Important

* Add environment variables in both platforms
* Update API URL after deployment
